# Decision Shortcuts

**When to read:** After your investigation is complete but before finalizing the verdict (Section 3). This guide distills patterns from 20+ real audits into quick decision rules that help you avoid common mistakes and spot obvious cases.

---

## 4 Decision Shortcuts (from 20+ real audits)

### 4.1 "All Links 404" = Fast REJECT (after Google Search)
When ALL provided URLs are dead AND Google Search in Chrome for username/name/email finds zero relevant results → Tier F + HIGH risk. Don't over-search — 3 failed Google searches is enough to conclude. But you MUST run the Google Search fallback before rejecting.

### 4.2 Dead Short Links — Normalize First
Short links expire but sellers don't. When a provided short link 404s, apply URL normalization (e.g., Whatnot `/invite/` → `/user/`) and visit the normalized URL in Chrome. If all normalized URLs also fail, proceed to Google Search fallback.

### 4.3 Category Mismatch Is Common (~15%)
~15% of sellers have mismatched categories. Always verify actual storefront content in Chrome before selecting the SOP. If mismatched, switch to the actual category's SOP.

### 4.4 Niche Hobbyist = Usually Legitimate
Small-scale niche sellers (isopods, rare plants, handmade crafts) with authentic content and low follower counts → typically Tier B + LOW risk = APPROVE. Low followers ≠ fraud, it means small market.

### 4.5 S-Tier Is Obvious
Consistent branding across 2-3+ platforms, high review counts with 4.9+ ratings, professional photography. If you have to debate it, they're not S-Tier.

### 4.6 Expired Website + Minimal Social = Tier F
"Account Expired" (Squarespace/Shopify) + <50 followers across all platforms → almost certainly inactive/abandoned.
