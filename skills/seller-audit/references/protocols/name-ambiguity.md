# Name Ambiguity Protocol

**When to read:** During the investigation phase (Section 2.5) when searching online uncovers records with names matching the applicant but you are unsure whether they refer to the same person. Use this protocol to systematically evaluate match confidence.

---

## 2.7.3 Name Ambiguity Protocol

If a high-risk record matches the applicant's name:
- Gold Match (Name + Age + Location + Email) → flag immediately
- Silver Match (Name + Location + Middle name) → flag immediately
- Bronze Match (Name + Location only, common name) → search for tie-breaker before flagging. If no tie-breaker, verdict must be REVIEW, not REJECT.
